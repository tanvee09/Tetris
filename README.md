## Try it out

```
$ git clone https://gitlab.com/tanvee_balhara/tetris.git 
$ cd tetris/server  
$ npm install  
$ nodemon index.js  
```
Go to [http://localhost:3000/tetris](http://localhost:3000/tetris)
